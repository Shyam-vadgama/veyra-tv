# Security Policy

## Supported Versions

Please check the latest release on GitHub to ensure you are using a supported version. We generally support the latest major release.

## Reporting a Vulnerability

We take the security of Veyra TV seriously. If you discover a security vulnerability, please **DO NOT** open a public issue.

Instead, please report it privately:

1.  Email us at **shyam.veyra.tv@gmail.com**.
2.  Include a description of the vulnerability and steps to reproduce it.
3.  We will acknowledge your report and work to fix it as soon as possible.

Thank you for helping keep Veyra TV safe!
